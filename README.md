# mi-ecommerce-con-Grid

Proyecto estático desarrollado por *Patricio Tapia Lavín* como práctica de maquetación web con HTML y CSS, enfocado en el uso de *CSS Grid* para la estructura principal y *Flexbox* para componentes específicos.

## 🚀 Acceso
👉 https://ptapialavin.github.io/mi-ecommerce-con-Grid/

## 📝 Descripción
La página simula una tienda en línea con tres secciones clave:
* *Navbar:* Encabezado centrado con el nombre de la tienda.
* *Sidebar:* Panel lateral izquierdo con categorías de productos y contadores.
* *Grilla de Productos:* Catálogo central distribuido en tarjetas con imágenes, descripción, precio y botón de compra.

## 🛠️ Tecnologías Utilizadas
* HTML5
* CSS3 (Grid Layout & Flexbox)

## ✨ Características
* Layout principal con división en dos columnas utilizando CSS Grid.
* Cuadrícula de productos modular con tres columnas automáticas.
* Tarjetas flexibles creadas con Flexbox para alinear el botón de compra al fondo.
* Efecto interactivo hover en los botones de compra.
